# Source Texts

This directory holds the manuscript files processed by the pipeline.
The texts are not distributed with this repository.

## How to obtain the texts

| Book | Source |
|------|--------|
| Meditations — Marcus Aurelius | [Project Gutenberg #2680](https://www.gutenberg.org/ebooks/2680) (public domain) |
| The Art of War — Sun Tzu (Giles trans.) | [Project Gutenberg #132](https://www.gutenberg.org/ebooks/132) (public domain) |
| Content — Cory Doctorow | [craphound.com/content/download](https://craphound.com/content/download/) (CC BY-NC-SA) |
| Being No One — Thomas Metzinger | MIT Press, 2003. Obtain via publisher or library. |
| The Fourth Culture — Rayan B. Vasse | Contact author. |

Download the texts and place them in this directory as:
- `meditations_marcus_aurelius.txt`
- `art_of_war_sun_tzu.txt`
- `doctorow_content.txt`
- `metzinger_being_no_one.pdf`
- `fc_iwb.docx`

Then run the pipeline with `python -m study.run_pipeline`.
